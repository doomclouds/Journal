结构校验失败

系统不会覆盖正式 Markdown。当前编辑会保存为 attention draft，并显示修复建议。

  • 缺少必需块：today-focus

    可以自动插入空的“今日重点”块，或回到块编辑模式手动补充。

  • 发现未知 section：random-note

    源码模式可见，但保存正式文件前需要改为 JMF 已知块或删除 marker。

  • marker 未成对

    系统可尝试按标题弱解析；弱解析失败时不允许确认写入。
