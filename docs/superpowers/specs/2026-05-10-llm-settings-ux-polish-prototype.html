1. 选择供应商

左侧直接看“已启用、已配置、需要配置、测试失败、备用”。

2. 编辑连接

Key 用密码式显示，来源清楚区分环境变量和配置文件。

3. 测试当前表单

编辑后旧测试结果过期，只测试当前输入，不偷换成已保存配置。

4. 保存并启用

自动测试成功才保存启用；失败保留输入，不污染当前可用配置。

5. 今日页重整理

内容变更回到今日页确认，设置页不直接改日记草稿。
